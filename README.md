## 📋 프로젝트 개요

**BEYOND REVIEW**는 릴스 마케팅 전문 에이전시 "담넘은담"의 공식 웹사이트입니다. 평균 조회수 4만, 최대 100만 이상의 실적을 가진 전문 크리에이터 팀이 맛집과 플레이스를 위한 릴스 마케팅 서비스를 제공합니다.

### 🎯 주요 목표
- 릴스 마케팅 서비스를 명확하게 소개
- 3가지 패키지(BASIC, STANDARD, PREMIUM) 안내
- 포트폴리오를 통한 실적 증명
- 간편한 문의 및 신청 시스템

## 📊 데이터 모델

### 데이터베이스: inquiries 테이블
신청 폼 데이터는 RESTful Table API를 통해 자동으로 저장됩니다.

```javascript
{
  id: String,               // 고유 ID (자동 생성)
  storeName: String,        // 매장 이름 (상호명)
  phone: String,            // 연락처 (010-0000-0000)
  email: String,            // 이메일 (선택)
  category: String,         // 촬영 종류
  package: String,          // 희망 패키지 (BASIC/STANDARD/PREMIUM)
  timing: String,           // 희망 촬영 시기
  message: String,          // 특별 요청사항
  privacy: Boolean,         // 개인정보 동의
  marketing: Boolean,       // 마케팅 수신 동의
  submittedAt: String,      // 신청 일시 (ISO 8601)
  created_at: Number,       // 생성 시간 (밀리초)
  updated_at: Number        // 수정 시간 (밀리초)
}
```

### RESTful API 엔드포인트
- **GET** `/tables/inquiries` - 모든 신청 목록 조회
- **GET** `/tables/inquiries/{id}` - 특정 신청 조회
- **POST** `/tables/inquiries` - 새 신청 생성
- **PUT** `/tables/inquiries/{id}` - 신청 전체 수정
- **PATCH** `/tables/inquiries/{id}` - 신청 부분 수정
- **DELETE** `/tables/inquiries/{id}` - 신청 삭제
